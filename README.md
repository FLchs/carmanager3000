## Development

<details open>
<summary>
Pre-requisites
</summary> <br />
    Car Manager 3000 is using [Bun](https://bun.com/) as package manager and runtime.
    To start developing you will need:

###

- Git
- Bun

</details>

To install dependencies:

```sh
bun install
```

To run:

```sh
bun run dev
```

Open <http://localhost:5173> in your browser.
